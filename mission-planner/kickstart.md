# kickstart guide  

## Windows  
```
install python3  
install pip3  
install Flask  
install flask_login  
install flask_bcrypt  
install flask_wtf
install wtforms  
install mysql.connecter  
install git  
login to git  
clone nisbweb/nisbconnect  
run the server python app.py  
check the website on browser  
make changes  
fill the form and submit  
```
